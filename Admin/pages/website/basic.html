Global Options

Notice: The values prefiexed with site_* are used to store details about the website and author, you may set your website title here.

Warning: The values here are very important and are used for several features of this website. Values prefiexed with default_* are used as default values for pages; other values can be used for global options, for example Google Analytics ID, or current stylesheet - changing these values can have significant effects on the stability of the website so take care when modifying them.
