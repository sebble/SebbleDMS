SebbleDMS

Sebble Data Management has been built from the ground up with security in mind, allowing fine grained controll over the roles and added permissions users may have.

If you are interested in the project you can check out our wiki over at GitHub (sebble/SebbleDMS). If you are having any problems with the system please leave a quick issue report also on GitHub (new issue).

SebbleCMS

SebbleCMS is a much older project which has been evolving for several years. Initially started as an simple webpage compiler, based on directory structure alone, this has now grown to incorporate many well known libraries such as Smarty 3 and has many advanced feature including a home-grown Wiki-markdown syntax, and now SebbleDMS.
